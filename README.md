# How to run :information_source:
```
# Clone this repository into your computer
$ git clone https://github.com/1TATO/twitter-clone.git

# Enter this repository
$ cs twitter-clone

# Run yarn to install all dependencies
$ yarn install

# Run the project
$ yarn start
```
